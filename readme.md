# git-secrets

store keys safely inside your git repositories.

## Installing

```
npm i -g git-secrets
```

### USAGE

```
git-secrets help
```

```
git-secrets init
```

environment variable supported: GIT_SECRETS_KEY, GIT_SECRETS_CONFIG
